# 赛博女友 - Cyber Girlfriend

基于 Live2D 虚拟角色的网页端 AI 伴侣应用。

## 项目定位

一款**情感陪伴型**虚拟交互产品，通过 Live2D 实时渲染的可爱形象 + AI 对话，为用户提供拟人化的聊天体验。

## 核心功能

| 功能 | 说明 |
|------|------|
| Live2D 形象展示 | 实时渲染的二次元虚拟女友（haru） |
| AI 对话 | 基于 MiniMax API 的自然语言交互 |
| 记忆系统 | 持久化聊天历史 + 情感状态 |
| 情感反馈 | 根据对话内容分析并更新角色情绪 |

## 用户画像

- **目标用户**：需要情感陪伴、喜欢二次元文化的年轻用户
- **使用场景**：碎片化时间闲聊、压力释放、孤独陪伴
- **核心价值**：24小时在线、永远有耐心、不会冷场的虚拟伴侣

## 产品特色

1. **有温度的形象** - Live2D 动态角色，不是静态图片
2. **会成长的关系** - 记忆系统让对话有连续性
3. **乖巧不打扰** - 主动聊天可控，避免骚扰

## 技术架构

```
前端 (Vite + PIXI.js)
    ↓ HTTP
后端 (Express.js)
    ↓ API
MiniMax (minimax-m2.7)
```

## 快速启动

```bash
# 安装依赖
npm install

# 启动后端（端口 3001）
node server.js

# 启动前端（端口 3000）
npm run dev
```

## 项目结构

```
├── server.js          # 后端服务
├── src/
│   ├── main.js       # 前端入口
│   └── chat.js       # 聊天逻辑
└── public/
    ├── models/       # Live2D 模型
    └── *.min.js      # 依赖库
```

## 注意事项

- API Key 配置在 `.env` 文件中
- 模型文件位于 `public/models/haru/`
